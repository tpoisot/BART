Overview of the types

BART.Tree — Type
Tree

A tree is a simple datastructure that binds a response y, a matrix of features X, and the tree itself stored in root. When the type is constructed without a root, the default behavior is to create a root that is a terminal node of depth 0, with a response set to the average of y.
